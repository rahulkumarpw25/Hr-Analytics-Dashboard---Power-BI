📊 HR Analytics Dashboard – Power BI Project

📌 Project Overview:

Employee attrition is a critical challenge for organizations, impacting productivity and costs. This project builds an HR Analytics Dashboard in Power BI to analyze workforce data, identify attrition drivers, and provide actionable insights for HR decision-making.

🛠 Tools & Technologies:

Power BI Desktop – Data modeling, visualization, and dashboard creation

Power Query – Data cleaning and transformation

DAX (Data Analysis Expressions) – Measures, KPIs, and calculations

CSV Dataset – HR employee records

🔄 Analytics Workflow:

Data Import – Load raw HR CSV files into Power BI

Data Cleaning – Handle missing values, datatype corrections (e.g., AttritionCount as Whole Number)

Data Modeling – Establish relationships between tables

KPI Creation – Attrition rate, average tenure, department-wise attrition

Visualization – Charts, tables, slicers, and filters for interactive exploration

Dashboard Publishing – Export and share insights with stakeholders

📈 Key Insights:

Attrition Trends – Higher attrition among younger employees and specific departments

Department Analysis – Sales and HR show elevated turnover compared to IT and Finance

Tenure Impact – Employees with <2 years tenure are more likely to leave

Demographics – Gender and age segmentation reveal distinct attrition patterns

Strategic Recommendations – Focus retention efforts on high-risk groups, improve onboarding, and strengthen engagement programs

🚀 Outcome:

The dashboard empowers HR teams to monitor attrition in real-time, identify root causes, and design targeted retention strategies. It demonstrates end-to-end Power BI capabilities, making it a strong addition to a data analyst portfolio.
